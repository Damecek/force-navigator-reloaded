Force Navigator Reloaded

A fast command palette for Salesforce Lightning. Search, launch, and navigate without leaving the keyboard.

Works only in Salesforce Lightning (not in Classic).

Get started in 60 seconds

  1. Open Lightning

    Navigate to any Lightning page in your org. Supported domains are force.com, salesforce-setup.com, and builder.salesforce-experience.com.

    Open your first Lightning tab

  2. Open the command palette

    Press Ctrl+Shift+L (Windows) or Cmd+Shift+P (Mac). Press the shortcut again or Esc to close it.

  3. Search and launch

    Type a command, object, flow, or setup page name and press Enter.

What it can do

Instant search

Fuzzy search with usage-based ranking for fast discovery.

Setup, objects, flows

Navigate setup, object metadata, flows, and admin pages quickly.

Customizable sources

Edit JSON settings to add custom commands or hide sources.
